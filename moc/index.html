Mile of Cry


Mile of Cry was a 3D point-n-click adventure game project for the PC with a psychological horror/thriller theme. I wrote a large part of the game code with the main focus on gameplay, controls, puzzle system, hint system, game object handling, game trigger handling and pathfinding. After the demo production was complete I also refactorized the majority of the game and did some preparations for its porting to other platforms. The demo was released only to the publishers, so for the general public only in-game screenshots are available. The project was eventually dropped due to a team break-up. You can still see some remnants of activity on the abandoned Facebook fanpage, though for obvious reasons it’s no longer updated.
